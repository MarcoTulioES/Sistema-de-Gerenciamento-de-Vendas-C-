# Sistema de Gerenciamento de Vendas em C

Projeto desenvolvido em linguagem C com o objetivo de simular um sistema simples de gerenciamento de vendedores. O sistema permite realizar operações básicas de cadastro e consulta de dados, utilizando manipulação de arquivos para armazenamento das informações.

Este projeto foi desenvolvido com foco no aprendizado de conceitos fundamentais de programação, incluindo estruturas de dados, manipulação de arquivos e lógica de programação.

---

## Funcionalidades

- Cadastro de vendedores  
- Listagem de vendedores cadastrados  
- Armazenamento de dados em arquivos  
- Interface de menu interativo no terminal  

---

## Tecnologias Utilizadas

- Linguagem C  
- Manipulação de arquivos  
- Estruturas (`struct`)  
- Lógica de programação  
- Programação estruturada

## Como Executar o Projeto

### 1. Clonar o repositório

```
git clone https://github.com/MarcoTulioES/Sistema-de-Gerenciamento-de-Vendas-C.git
```

### 2. Entrar na pasta do projeto

```
cd Sistema-de-Gerenciamento-de-Vendas-C
```

### 3. Compilar o programa

```
gcc main.c -o sistema
```

### 4. Executar o programa

Linux / Mac:

```
./sistema
```

Windows:

```
sistema.exe
```
